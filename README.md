# CS2100-F25

passwork

project description
passwork website that helps people make strong passwords and stay secure online. it has a lot of tools like a password strength tester , a customizable password generator, and some resources about keeping your accounts secure.

installation instructions
no installation needed.

usage instructions
home page: see what the site has to offer
password strength tester: type in a password and see how "strong" it is and how long it might take to crack
password generator: pick options like length, symbols, numbers, and generate a secure password
resources: guides and tools
contact: send feedback/questions through the contact form

all pages have nav links at the top.
